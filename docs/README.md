### Foundation (Storybook)

Storybook에서 확인 가능한 디자인 시스템 문서:

- **Overview** - 디자인 시스템 개요 및 핵심 특징
- **Colors** - 색상 팔레트 및 토큰 시스템
- **Typography** - 폰트, 크기, 무게, 행간
- **Spacing** - 간격 시스템
- **Accessibility** - 접근성 가이드 및 WAI-ARIA 표준
